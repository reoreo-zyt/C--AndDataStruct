## 1. 总结

1. c++ 程序由一个或者多个函数模块组成，程序由 `main()` 函数开始执行。

2. c++ 语句包括：

    * 声明语句
    * 赋值语句
    * 消息语句
    * 函数调用
    * 函数原型
    * 返回语句

3. 类是用户定义的数据类型规范，描述了如何表示信息以及可对数据执行的操作。

4. 对象是根据类规范创建的实体。如简单变量是根据数据类型描述创建的实体。

5. `cin`、`cout` 是处理输入输出的自定义对象。是 `istream` 和 `ostream` 类的实例。这两个类是在 `iostream` 中定义的。

6. 使用库函数，应当包含提供函数原型的头文件。
